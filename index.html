              # A simple Python function to calculate future value
              def calculate_future_value(principal, rate, years):
                  future_value = principal * (1 + rate) ** years
                  return future_value

              # Example usage
              initial_investment = 10000
              annual_rate = 0.05

              final_value = calculate_future_value(initial_investment, annual_rate, time_in_years)

Valuation Models

The **Discounted Cash Flow (DCF)** formula is a key valuation method:

$$ DCF = \sum_{t=1}^{n} \frac{CF_t}{(1+r)^t} $$

Where:

  • $CF_t$ = Cash flow in year t
  • $r$ = Discount rate
  • $t$ = Time period
Here we show the DCF formula. This demonstrates how we calculate the present value of future earnings to assess a company's worth.
